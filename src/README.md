# 📦 新一代电商退货截图模拟器 (React版)

[![GitHub stars](https://img.shields.io/github/stars/zgytlkj/sf-express-return-picture.svg?style=social&label=Star)](https://github.com/zgytlkj/sf-express-return-picture)

## 简介

本项目是一个基于 **React + TypeScript + Tailwind CSS** 构建的高度仿真的电商平台退货详情页面。它的核心目的是帮助用户快速生成一张逼真的退货凭证截图，以便在顺丰、京东等快递平台小程序中上传，从而享受专为电商用户提供的寄件优惠。

相比原有的纯HTML版本，此版本在保留核心功能的基础上，进行了全面的技术重构和功能升级，提供了更流畅、更智能的用户体验。

## ✨ 核心功能

- **现代化技术栈**: 使用 React, TypeScript, 和 Tailwind CSS 构建，确保代码质量和可维护性。
- **高仿真UI**: 精准复刻主流电商平台的退货详情页面，包括状态流程、寄件信息、服务详情等模块。
- **智能地址识别**: 支持从剪贴板一键粘贴收货信息，并能自动解析填充“收货人”、“电话”和“详细地址”字段。
- **一键生成截图**: 内置截图功能，点击“我要寄件”即可将整个页面保存为一张高清图片，无需手动截图和裁剪。
- **完全在浏览器端运行**: 所有操作均在您的浏览器中完成，不依赖任何后端服务，保障您的隐私安全。

## 🚀 使用方式

### 在线使用

1.  访问项目在线地址: **https://sf-exp.pages.dev/**
2.  在“寄回商品”区域，您可以手动填写收件信息。
3.  **（推荐）** 更快捷的方式是：先复制包含姓名、电话和地址的文本，然后点击输入框旁边的 **“复制”** 按钮，应用会自动为您解析并填充表单。
4.  信息确认无误后，点击橙色的 **“我要寄件”** 按钮。
5.  浏览器将自动生成并下载一张名为 `退货退款详情.png` 的截图。
6.  打开顺丰、京东或其他快递平台的小程序/APP，在寄件时选择“图片识别地址”或“上传凭证”，上传刚刚保存的图片即可享受优惠。

### 本地开发

如果您想在本地运行或进行二次开发：

```bash
# 克隆仓库
git clone https://github.com/zgytlkj/sf-express-return-picture.git

# 进入项目目录
cd sf-express-return-picture

# 安装依赖 (推荐使用 pnpm)
pnpm install

# 启动开发服务器
pnpm dev
```
